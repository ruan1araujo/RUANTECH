## 🌐 Deploy no GitHub Pages

### **Método Automático (Recomendado)**

O projeto já inclui um workflow GitHub Actions (`.github/workflows/pages.yml`) que faz deploy automático:

1. **Push para main** - Deploy automático é acionado
2. **Aguarde ~2 minutos** - Processo de build e deploy
3. **Acesse** - `https://seu-usuario.github.io/RUANTECH/`

### **Método Manual**

1. Vá em **Settings → Pages** no seu repositório
2. Em **Source**, selecione `Deploy from a branch`
3. **Branch**: `main` | **Folder**: `/ (root)`
4. Clique em **Save**
5. Aguarde alguns minutos e acesse o link gerado

### **Verificar Status**

- **Actions**: Veja o progresso na aba "Actions" do repositório
- **Settings → Pages**: Link da página publicada aparece aqui
